## Goal

The goal of this exercise is to teach the student about symbols and tuples in Crystal.

## Learning objectives

- Know how to create a tuple
- Know how to get items from tuple
- Know how to define a symbol
- Know when you can use symbols

## Out of scope

- Named tuples

## Concepts

`Tuples`:

- Create a Tuple
- Index tuples

`Symbol`:

- Create symbol
- Compare symbol

## Prerequisites

- Iteration
